Security

인증 & 보안

인증 시스템

  • JWT 액세스 + 리프레시 토큰
  • bcrypt 패스워드 해싱
  • 역할 기반 접근 제어 (RBAC)
  • 로그인 이력 추적 (IP, UA)
  • 계정 잠금 (5회 실패 시)

보안 기능

  • IP 블랙리스트 (CIDR 지원)
  • 파일 업로드 확장자 차단
  • CORS 정책 설정
  • 요청 로깅 & 모니터링
  • 슬로우 쿼리 감지 & 알림
